Q.how to use css

A.There are three ways to use css
    1. Inline css:      inside a starting tag we use Style attribute and in that we specify property and value.
        syntax:         style="property:value;"
    2. Internal css:    in html file we use Stule tah and in that we select element apply css property.
        syntax:         selector{
                            property:value;
                            property:value;
                            property:value;
                        }
    3. External css:    in this we create a seperate css file and import that css in header part of html file 


*selector in css : Selecting an element
1. Element or tah name selector : In this we use element or tah name to aply css property
2. Group selector : in this we make group selector to apply css property for multiple elements
3. Id selector : if you have to apply css property to an single element in that case we use it
4. Class Selector : if you have to apply a specific css property to a bunch of elements
5. Universal Selector : if you have to apply css property to all elements



*border : a line drawm around the element


*Box-Model : it contain four point
1. content :    it can be any image or text
2. padding :    it is used to clear area between conent and element border
3. border :     a line drawn around and element
4. margin :     it is used to clear area around the element border.


*position:
- it is used to move element after applying left, right, top and bottom property value
- by default every element has position:static
1. position: static(default)
2. position: relative; element will move from left, right, top and bottom side
3. position: absolute; when we wan tto move child element according to its parent element
4. position: fixed; when we have to fixed on screen window
5. position: sticky; when we have to stick element on screen window
